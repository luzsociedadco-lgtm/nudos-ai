# Nudos AI

Nudos AI is an AI-powered application designed to verify and analyze real-world sustainability actions.

## Purpose
This repository demonstrates a production-ready AI workflow using DigitalOcean Gradient™ AI, focused on transparency, verification, and scalable deployment.

## Tech Stack
- Python
- FastAPI
- DigitalOcean Gradient™ AI
- Docker (optional)

## How it works
Users submit data related to sustainability actions. The system analyzes inputs and returns a verification score and impact metrics.
